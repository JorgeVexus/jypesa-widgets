# Changelog - 2026-07-07

## Resumen de la Sesión
En esta sesión se implementaron tres widgets interactivos y adaptables para Jypesa (Hero, Línea de Tiempo y Tarjetas de Capacidades), adaptándolos para integración en Webflow y configurando su despliegue continuo en Vercel.

## Cambios Detallados

### Configuración del Proyecto
*   **vercel.json**: Creado para habilitar cabeceras CORS en todas las rutas (`Access-Control-Allow-Origin: *`), permitiendo que el script sea importado desde Webflow.
*   **.gitignore**: Configurado para ignorar directorios de configuración de IDEs (`.claude`, `.codex`, `.gemini`) y otros archivos innecesarios.
*   **AGENTS.md**: Creado bajo la carpeta `.agents/` con reglas de comportamiento para el entorno local.

### Widget Hero Interactivo
*   **widgets/hero-interactive/hero-widget.js**: Script auto-contenido de inyección (IIFE) que carga tipografías, estilos CSS de forma dinámica y genera el HTML del Hero. Implementa una animación fluida en 3 fases:
    *   *Fase 1*: Imagen Hero a pantalla completa con efecto de escala suave.
    *   *Fase 2*: Entrada deslizante del texto "Sobre" (desde arriba hacia abajo) y el logotipo de Jypesa (desde abajo hacia arriba) alineados adecuadamente.
    *   *Fase 3*: Reducción de la imagen superior para revelar un carrusel de marcas y un grid con 4 bloques de estadísticas (KPIs).
*   **widgets/hero-interactive/hero-widget.css**: Contiene todos los estilos, animaciones personalizadas de flujo de gradiente para los números y las directivas de responsividad (media-queries).
*   **hero.html**: Creado como panel local de vista previa y depuración del widget, incluyendo una utilidad javascript para reiniciar y probar la animación de forma manual.

### Widget de Línea de Tiempo
*   **widgets/timeline/timeline-widget.js**: Script modular (IIFE) que maneja el marcado HTML, transiciones y un sistema de autoplay con detención por interacción.
*   **widgets/timeline/timeline-widget.css**: Estilos y keyframes de carga progresiva acelerada. La línea de progreso activa está centrada verticalmente con los dots y no muestra fondo gris inactivo al inicio.
*   **timeline.html**: Panel de visualización local y depuración para la línea de tiempo con opción de reinicio rápido.

### Widget de Tarjetas de Capacidades
*   **widgets/cards/cards-widget.js**: Script dinámico (IIFE) que inyecta el marcado para las tres tarjetas (Producción, Private label, Licencias), cargando tipografías e imágenes personalizadas del cliente.
*   **widgets/cards/cards-widget.css**: Hojas de estilo que manejan el grid responsivo, el zoom suave de la imagen (`scale(1.04)`), la elevación del contenedor (`translateY(-8px)`) y la animación del chevron del botón (desplazamiento a la derecha de `6px`).
*   **cards.html**: Panel de vista previa y depuración local para ensayar los efectos hover.
