# Changelog — 2026-07-19

## feat: implement dispenser comparison grid widget with mobile slider, dynamic categories and webflow integration

### Descripción
Se desarrolló e implementó el nuevo widget interactivo **Sistemas de Dispensación Comparativa** (Grid de Comparación) basado en las especificaciones de diseño de Figma. Admite carga dinámica a través de la API del CMS de Webflow, cambio de imágenes principales al hacer clic sobre miniaturas, navegación mediante deslizamiento horizontal (*swipe*) con dots indicadores en dispositivos móviles, y filtrado dinámico de secciones mediante nombres de categorías y descripciones provistos desde el CMS de Webflow.

### Archivos creados
- **`widgets/sistemas-dispensacion-comparativa/sistemas-dispensacion-comparativa.js`** — Script principal del widget con lógica de inyección de estilos CSS, cargado dinámico/fallback, mapeo de miniaturas variantes, efectos de transiciones de imagen, y dots navegacionales del slider móvil.
- **`widgets/sistemas-dispensacion-comparativa/preview.html`** — Panel de previsualización local para pruebas de carga dinámica por CMS simulado y fallback estático con filtros de categorías.
- **`widgets/sistemas-dispensacion-comparativa/README.md`** — Documentación técnica de integración, nombres de campos CMS requeridos, asignación de clases CSS en Webflow y código embed HTML.

### Cambios detallados
- **Layout de Figma:** Cuadrícula de 4 columnas en desktop con líneas divisorias verticales, tipografía Montserrat y Rubik, bordes y colores slate (#506D85), verde (#4aa25d) y azul (#48a9c5) de acuerdo al diseño.
- **Galería Interactiva:** Al hacer clic en las miniaturas de productos compatibles, la imagen principal se reemplaza suavemente usando una animación de cross-fade (200ms de opacidad) y el elemento activo se resalta con un borde de color destacado.
- **Filtrado por Categorías:** Mapeo de campos `.jypesa-disp-categoria` y `.jypesa-disp-categoria-desc` desde Webflow CMS. Permite a los bloques embed cargar y nombrar dinámicamente secciones completas usando el atributo `data-category-filter` (ej. "Sistemas de dispensación", "Soportes", "Complementos").
- **Slider Responsivo Móvil:** En resoluciones de pantalla iguales o menores a 768px, el grid se adapta automáticamente a un carrusel de desplazamiento horizontal que muestra un producto por slide. Adiciona puntos navegacionales dinámicos en la base que se sincronizan de forma bidireccional con el scroll.
