# Changelog — 2026-07-17

## feat: Implementación del widget Sistemas de Dispensación (Scroll Split)

### Descripción
Se creó e implementó el widget dinámico "Sistemas de Dispensación" con un efecto de desplazamiento vertical opuesto (scroll split) de alto impacto visual adaptado de Figma. En pantallas de escritorio, las imágenes de los laterales se desplazan hacia arriba y el bloque de texto central desciende a medida que el usuario hace scroll. Soporta datos locales de fallback y lectura desde una colección CMS en el DOM de Webflow.

### Archivos modificados
- **`widgets/sistemas-dispensacion/sistemas-dispensacion.js`** — Código principal del widget con lógica de scroll interactivo sticky en desktop, inyección de estilos premium (degradados en títulos y cursivas estilizadas) y adaptabilidad móvil como cuadrícula responsiva estática.
- **`widgets/sistemas-dispensacion/preview.html`** — Panel de pruebas local con simulación de fuentes de colección CMS y datos locales.
- **`widgets/sistemas-dispensacion/README.md`** — Documentación técnica para la instalación del widget de scroll split en Webflow.

## feat: Implementación del widget de Carrusel Infinito (Partners Marquee)

### Descripción
Se implementó el widget de carrusel de marcas/socios con desplazamiento horizontal continuo e infinito. Incorpora lógica inteligente en JavaScript para autoduplicar los logos del CMS de Webflow (multiplicándolos si la cantidad es pequeña), garantizando un loop continuo y sin espacios en blanco en cualquier resolución de pantalla. Permite múltiples instancias con diferentes duraciones en la misma página leyendo una sola colección CMS.

### Archivos creados
- **`widgets/partners-marquee/partners-marquee.js`** — Lógica del widget con inyección de estilos responsivos, animación infinita linear y script de multiplicación inteligente de imágenes.
- **`widgets/partners-marquee/preview.html`** — Panel de pruebas local con múltiples duraciones de ciclo y simulación CMS con cantidad de logos reducida.
- **`widgets/partners-marquee/README.md`** — Guía de instalación y mapeo de clases para el Collection List de Webflow.
