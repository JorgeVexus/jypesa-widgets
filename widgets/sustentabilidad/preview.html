Para insertar este bloque interactivo a dos columnas en Webflow u otro proyecto HTML, agrega el contenedor con el ID correspondiente y carga el script asíncrono:

<!-- Script del Widget (Vercel) -->
<script src="https://jypesa-widgets.vercel.app/widgets/sustentabilidad/sustentabilidad-widget.js"></script>
