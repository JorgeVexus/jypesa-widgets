Integración en Webflow

Para insertar este widget de colecciones con sliders en tu proyecto de Webflow, sigue estos pasos:

Paso 1: Agregar el bloque del Widget (Embed)

Inserta un componente Embed en el lugar donde desees que aparezca el selector e inserta el siguiente contenedor:

<!-- Contenedor para el Widget de Colecciones -->
<div id="jypesa-tabs-colecciones-widget"></div>

Paso 2: Enlazar tu CMS Collection List en Webflow

Crea una Collection List (de tus productos de colección o amenidades) en la misma página (puede estar oculta, por ejemplo con style="display: none;" o display: none en Webflow). Asegúrate de agregar las siguientes clases CSS a cada elemento correspondiente:

  • Wrapper de la lista entera: jypesa-tabs-colecciones-cms-source
  • Nombre de la Colección (Texto): jypesa-tabs-col-name
  • Sub-Categoría / Variante (Texto, Opcional): jypesa-tabs-subcol-name
  • Logo / Submarca de la Colección (Imagen, Opcional): jypesa-tabs-col-logo
  • Familia Olfativa (Texto, Opcional): jypesa-tabs-col-familia
  • Mood de la fragancia (Texto): jypesa-tabs-col-mood
  • Indicador de Relleno ("RELLENABLE" / "NO RELLENABLE"): jypesa-tabs-col-refill
  • Nota de Salida (Texto, Opcional): jypesa-tabs-col-notes-salida
  • Nota de Corazón (Texto, Opcional): jypesa-tabs-col-notes-corazon
  • Nota de Base (Texto, Opcional): jypesa-tabs-col-notes-fondo
  • Descripción de la Colección (Texto): jypesa-tabs-col-desc
  • Nombre del Producto (Texto): jypesa-tabs-prod-name
  • SKU (Texto): jypesa-tabs-prod-sku
  • Peso/Volumen (Texto): jypesa-tabs-prod-weight
  • Tipo de Empaque (Texto): jypesa-tabs-prod-packaging
  • Cantidad por Caja (Texto): jypesa-tabs-prod-qty
  • Imagen del Producto (Imagen): jypesa-tabs-prod-img
  • Link del Producto (Link Block): jypesa-tabs-prod-link
  • Link de Amazon (Link Block / Texto URL, Opcional): jypesa-tabs-prod-amazon-link

Paso 3: Cargar el Script

Al final de la página (en el panel Custom Code antes del cierre de </body>), agrega la llamada al script de producción:

<!-- Enlace del Script del Widget (Vercel) -->
<script src="https://jypesa-widgets.vercel.app/widgets/tabs-colecciones-productos/tabs-colecciones-productos.js"></script>
