📦 Integración en Webflow — Hotelería Tabs Widget

Widget de tabs de dos columnas, pixel perfect al diseño Figma node-id=469-25717. Muestra segmentos hoteleros con imágenes cambiables por CMS.

Paso 1 — Agregar el contenedor del widget

Inserta un bloque Embed donde quieres el widget:

<div id="jypesa-hoteleria-tabs-widget"></div>

<!-- O con clase (permite múltiples instancias) -->
<div class="jht-widget-container"></div>

<!-- O con atributo personalizado -->
<div data-widget="hoteleria-tabs"></div>

Paso 2 — Crear la Collection List del CMS (oculta)

Agrega una Collection List de Webflow en la misma página con display: none. Asigna la clase jht-cms-source al wrapper, y las siguientes clases a cada elemento:

Clase CSS Elemento Webflow Descripción Tipo
jht-cms-source Collection List Wrapper Wrapper de toda la lista. Ponlo con display:none Requerido
jht-cms-segment-label Text Block (en el wrapper, no en cada ítem) Etiqueta de segmento, p. ej. "Segmento • Hotelería" Opcional
jht-cms-section-title Text Block (en el wrapper) Título grande, p. ej. "Hotelería" Opcional
jht-cms-section-desc Text Block (en el wrapper) Descripción general de la sección Opcional
jht-cms-tab-label Text Block (en cada w-dyn-item) Nombre del tab / segmento. Cada ítem = un tab Requerido
jht-cms-tab-desc Rich Text / Text Block (en cada ítem) Descripción que aparece en el panel derecho Requerido
jht-cms-tab-img Image (repetir hasta 3 veces en cada ítem) Imágenes del panel. Se muestran máximo 3 por tab Requerido

Paso 3 — Cargar el script

Al final de la página (antes de </body>), en el panel de Custom Code:

<script src="https://jypesa-widgets.vercel.app/widgets/hoteleria-tabs/hoteleria-tabs.js"></script>

Múltiples instancias con CMS diferente

Usa el atributo data-cms-source para apuntar explícitamente a una fuente CMS diferente:

<div class="jht-cms-source hospitalidad" style="display:none">
  <!-- Items de hospitalidad avanzada -->
</div>

<div class="jht-widget-container" data-cms-source=".jht-cms-source.hospitalidad"></div>
